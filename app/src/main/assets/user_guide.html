First-Time Setup

1 Install APK — Download from GitHub Releases or via "Check Updates" button

2 Enable Accessibility Service

Settings → Accessibility → OpenClaw → ON

This lets Nate see and interact with your screen (tap, swipe, type, read UI)

3 Grant Full Storage Access

OpenClaw Settings → Permissions → Grant "Full Storage Access"

Lets Nate access files on /sdcard/ (Photos, Downloads, Documents)

4 Add LLM Provider (or use free default)

OpenClaw Settings → LLM Providers

  • FREE Pollinations — works immediately, no key needed
  • FREE Gemini — get free key at aistudio.google.com
  • PAID OpenRouter — one key, 100+ models

5 Start Service

OpenClaw Settings → tap "Start Service"

6 Start chatting!

Go to Chat tab → type a message or use /status to verify

After APK Updates

Re-enable Accessibility Service after every update — Android disables it automatically.

Settings → Accessibility → OpenClaw → ON

Your data is safe! Chat history, memories, scheduled tasks, API keys, and workspace files all survive APK updates.

Troubleshooting

Nate not responding?

Check: Service started? Accessibility enabled? LLM provider configured? Try /status

Can't find an app?

Nate searches by name. Say: "buka Pluang" not "buka com.EmasDigi"

Flutter apps (0 elements)?

Nate uses Gemini Vision instead of accessibility tree. Needs Gemini API key in Settings.

Python/Node won't install?

Some devices block binary execution (SELinux). The Linux environment (PRoot) may work as alternative.

Battery drain?

Settings → Battery → OpenClaw → "Unrestricted". Samsung: disable "Put unused apps to sleep".
