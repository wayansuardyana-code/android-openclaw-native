Google Workspace Setup

One-time setup ~ 5 minutes

Why? OpenClaw needs an OAuth2 token to create Google Sheets, send Gmail, manage Calendar events, etc. Reading files via the Google apps on your phone works without this — but creating/editing needs API access.

Step 1: Create a Google Cloud Project

1 Open console.cloud.google.com

2 Click "Select a project" (top bar) → "New Project"

3 Name it OpenClaw → Click Create

4 Make sure the new project is selected in the top bar

Step 2: Enable Google APIs

1 Go to APIs & Services → Library

2 Search and Enable each of these:

  • Google Drive API
  • Google Sheets API
  • Gmail API
  • Google Calendar API
  • Google Docs API

Click each one → click "Enable" → go back and search the next one

Step 3: Create OAuth2 Credentials

1 Go to APIs & Services → Credentials

2 Click "+ Create Credentials" → "OAuth client ID"

3 If asked to configure consent screen:

  • User type: External → Create
  • App name: OpenClaw
  • User support email: your email
  • Developer contact: your email
  • Click Save and Continue through all steps
  • Add your own email as a Test user

4 Back to Credentials → Create OAuth client ID:

  • Application type: Web application
  • Name: OpenClaw Android
  • Authorized redirect URIs: add https://developers.google.com/oauthplayground
  • Click Create

5 Copy the Client ID and Client Secret — you'll need them next

Step 4: Get an Access Token

1 Open OAuth 2.0 Playground

2 Click the gear icon (top right) → check "Use your own OAuth credentials"

3 Paste your Client ID and Client Secret

4 In the left panel, select these scopes:

  • Google Drive API v3 → select ../auth/drive
  • Google Sheets API v4 → select ../auth/spreadsheets
  • Gmail API v1 → select ../auth/gmail.modify
  • Google Calendar API v3 → select ../auth/calendar
  • Google Docs API v1 → select ../auth/documents

5 Click "Authorize APIs" → sign in with your Google account → Allow

6 Click "Exchange authorization code for tokens"

7 Copy the Access Token

Access Token Expires!

Access tokens expire after 1 hour. You'll need to refresh it periodically. Copy the Refresh Token too — future versions of OpenClaw will auto-refresh.

Step 5: Add Token to OpenClaw

1 Go to Settings in OpenClaw

2 Tap "+" to add a new service

3 Service name: google_oauth

4 Paste your Access Token

5 Tap Save

Done! Try It

Ask OpenClaw:

"Create a new Google Sheet called 'Budget 2026'"
"Send an email to boss@company.com about the meeting tomorrow"
"What's on my calendar today?"
"List my Google Drive files"

No token needed for reading! OpenClaw can read your Google Sheets, Docs, and Drive files by opening the apps directly on your phone. The OAuth token is only needed for creating and editing via API.
